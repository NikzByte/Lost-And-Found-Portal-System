# 🎒 BatStateU Lost and Found Listing System  

The **BatStateU Lost and Found Listing System** is a Python-based console application that allows students to report, search, and manage lost and found items in an organized way.

---

## 🎓 Project Overview

The **Lost and Found Listing System** is designed to address the common issue of misplaced items among students. Instead of relying on unofficial platforms like Facebook, this system provides a more organized and efficient way to track and retrieve items.

The project demonstrates practical application of programming concepts such as data handling, file storage, and basic searching algorithms.

---

## 🧪 Features and Functionalities

| Feature | Description |
|--------|------------|
| Post Lost Item | Allows users to report missing items |
| Post Found Item | Allows users to report found items |
| View Listings | Displays all stored items |
| Search Items | Enables keyword-based searching |
| Unique ID System | Generates unique identifiers per listing |
| File Storage | Saves and retrieves data using JSON |

---

## 🔄 System Workflow

| Step | Process |
|------|--------|
| 1 | User selects Lost or Found |
| 2 | Inputs item details (name, description, location, etc.) |
| 3 | System generates a unique ID |
| 4 | Data is saved to `listings.json` |
| 5 | Users can view or search listings |
